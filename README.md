# DASHBOARD-com-DJANGO-e-CHART.JS-


## Sobre o projeto

DASHBOARD para  análise de dados do desenvolvimento de uma empresa: será analisado o faturamento, despesas,
produtos mais vendidos e o funcionário do mês no período de 1 ano.
